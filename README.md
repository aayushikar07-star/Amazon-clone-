# 🛒 Amazon Clone

A responsive front-end clone of the Amazon website built using **HTML** and **CSS**. This project replicates the design and layout of Amazon's homepage to strengthen front-end web development skills and gain hands-on experience with responsive UI design.

## 🚀 Features

- Responsive navigation bar
- Search bar interface
- Hero banner section
- Product category cards
- Multiple product sections
- Footer with quick links
- Responsive layout for different screen sizes

## 🛠️ Tech Stack

- HTML5
- CSS3

## 📂 Project Structure

```
Amazon-Clone/
│── index.html
│── style.css
│── images/
└── README.md
```

## 🎯 Learning Outcomes

- Built a responsive web page using HTML and CSS
- Practiced Flexbox for layout design
- Improved CSS styling and positioning techniques
- Enhanced understanding of responsive web design
- Recreated a real-world e-commerce website interface
